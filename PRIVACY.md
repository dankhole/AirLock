# Privacy Policy

**Airlock** does not collect, transmit, or store any personal data.

All data (your tracked site list, delay duration, and enabled/disabled state) is stored locally on your device using the browser's built-in extension storage APIs. Timer session data is stored in session storage and is automatically cleared when the browser closes.

No data is sent to any server, third party, or external service. The extension makes no network requests. There is no analytics, telemetry, or tracking of any kind.

## Permissions

| Permission | Why |
|---|---|
| `storage` | Save your settings and timer state locally on your device |
| `activeTab` | Read the current tab's URL when you open the popup, so the "Track this site" button can work |

## Contact

If you have questions about this privacy policy, open an issue on the GitHub repository.
